# MeltingStudents
Does heat lead to fewer students staying at Loras through the first few weeks?


In this assignment, use the data provided, as well as temperature data from the web, to try to answer the question above.

Reading the chapters on [Data Import](https://r4ds.had.co.nz/data-import.html) and [Relational Data](https://r4ds.had.co.nz/relational-data.html) will be helpful and you **should** do it.

Using [`rwunderground`](http://rpubs.com/lefkios_paikousis/weatherdata-in-r) to get weather data will make your life easier.A line I wrote on my local computer
